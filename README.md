# 🍪 CookieMice Assistant

CookieMice Assistant is a Full-Stack AI-powered recipe assistant built with a separate Frontend and Backend architecture.

> ⚠️ **Important Notice**
>
> This repository does **NOT** include the required runtime assets, API secrets, and model files.
> These files are distributed separately by the project developer.

# 📦 Required Assets

Before running the project you must obtain the **Project Vitals** package from the developer.

Extract the archive exactly as shown below.

```
CookieMice-Assistant/
│
├── ASSETS/
│   └── Models/
│
├── Backend/
│
└── Frontend/
```

Do **NOT** rename folders.

Do **NOT** change directory names.

---

# 🔐 Project Vitals Package

The Vitals package contains all runtime resources required for the application.

Contents include:

- Runtime Models
- AI Assets
- Configuration Files
- Environment Variables
- API Request Collections
- Supporting Runtime Files

> The archive is password protected.

Please contact the project developer to obtain:

- The Vitals archive (.rar)
- Archive password
- Required API credentials (if applicable)

# ⚙️ Backend Setup

```bash
cd Backend
npm install
```

Create or place the provided `.env` file inside the Backend directory.

Start the backend:

```bash
npm start
```

or

```bash
node index.js
```

---

# 💻 Frontend Setup

Navigate to the frontend directory.

Install dependencies.

```bash
npm install
```

Run the development server.

```bash
npm run dev
```

---

# 🐳 Docker

Backend Docker support is included.

```bash
docker compose up --build
```

---

# ⚠️ Missing Files

If you encounter errors such as:

- Missing Models
- Missing Environment Variables
- API Authentication Errors
- Runtime Asset Errors

You most likely have **not installed the Project Vitals package**.
